# WebTools Dashboard

A modern, secure browser-based dashboard for simple web tools.

## Features
- Secure File Transfer tool
- Responsive, clean design
- Modern color palette and typography
- Footer with contact info

## Getting Started
1. Clone the repository.
2. Place your SVG logo in `src/media/logo.svg`.
3. Open `src/index.html` in your browser.

## Folder Structure
```
webtools/
├── src/
│   ├── index.html
│   ├── style.css
│   ├── media/
│   │   ├── logo.svg
│   │   ├── icon-transfer.png
│   │   └── icon-placeholder.png
│   └── ...
└── README.md
```

## Customization
- Edit `src/index.html` to add or remove tools.
- Update `src/style.css` for colors, layout, and branding.
